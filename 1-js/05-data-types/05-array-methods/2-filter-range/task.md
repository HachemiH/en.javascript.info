La fonction ne doit pas modifier le tableau. Elle doit juste retourner le nouveau tableau.

Par exemple:

```js
let arr = [5, 3, 8, 1];

let filtered = filterRange(arr, 1, 4);

alert( arr ); // 5,3,8,1 (non modifié)
```
